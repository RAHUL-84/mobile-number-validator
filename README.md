# Mobile Number Validator

## Project Overview
This is a **Mobile Number Validator** built using **Python, Django, and Django REST Framework**.  
The project allows users to validate **10-digit mobile numbers** through a **simple HTML & JavaScript interface**.  
All backend functionality is tested using **Django admin panel**.

---

## Technologies Used
- **Programming Language:** Python  
- **Framework:** Django, Django REST Framework  
- **Database:** SQLite  
- **Frontend:** HTML, JavaScript  
- **Tools:** VS Code  

---

## Features
- Validate 10-digit mobile numbers  
- Ensures number length is exactly 10 digits  
- Simple frontend interface with input box and **Validate button**  
- Validation results displayed dynamically on the page  
- Admin panel for managing backend data
